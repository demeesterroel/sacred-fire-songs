# Documentation Changelog Policy
- **Tracebility** Each document in /doc should contain a changelog section at the top of the file
- **Type-specific** In .md documents the changelog is part of the document, for .html and .sql files the changelog is in a comment section
- **Content** The changelog section has following format
**Version:** 1.13
**Status:** Draft
**Date:** December 24, 2025
